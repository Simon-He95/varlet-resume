# VARLET RESUME

A resume template based on [Varlet](https://github.com/varletjs/varlet)

## Try it Now!

### Github Template

[Create a repo from this template on GitHub.](https://github.com/varletjs/varlet-resume/generate)

### Clone to local

```
npx degit varletjs/varlet-resume my-resume
cd my-resume
pnpm i
```

### Usage

#### Development

```
pnpm dev
```

#### Build

```
pnpm build
```

#### Preview

```
pnpm preview
```
